# ac-twitter-api

## Project setup
```
git clone https://github.com/tommy0311/twitter-api-2020.git

cd twitter-api-2020

npm install

cp .env.example .env
(edit .env to set environment)
```

### Compiles and hot-reloads for development
```
npm run dev
(api server is running on http://localhost:3000/api)
```
