# Twitter 專案
### 環境需求

* Node.js 14.16.0 以上
* MySQL 8.0.28  
  <br />
### 本地安裝方式
  <br />

#### node.js
至[官網下載](https://nodejs.org/en/)

#### MySQL 
至[官網下載](https://dev.mysql.com/)
  <br />


## 安裝步驟
下載專案  <br />
```git clone https://github.com/eruc1117/twitter-api-2020.git```

移動至專案資料夾  <br />
```cd twitter-api-2020```

套件下載  <br />
```npm install```

補上環境變數  <br />

JWT_SECRET=自行設定  <br />
IMGUR_CLIENT_ID=自己在[imgur](https://api.imgur.com/)的ID

伺服器運行  <br />
```npm run dev```
