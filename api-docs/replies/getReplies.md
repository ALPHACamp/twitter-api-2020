# `GET` /api/tweets/:tweet_id/replies

## API feature
Get replies of a certain tweet.
* Ordered by timestamp of reply created

## Input data  
### parameters  
| params     | Description |
| ---------- | ----------- |
| `tweet_id` | tweet id    |
### req.body  
None
