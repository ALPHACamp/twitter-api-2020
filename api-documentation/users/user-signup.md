# `POST` /api/users
### Feature

使用者註冊一個新帳號

### URI Parameters

N/A

### Request Header

不須帶JWT(還沒登入)

```
content-type: application/json
```

### Request Body

| Name | Required | Description | Type |
| --- | --- | --- | --- |
| account | True | 使用者帳號 | String |
| name | True | 使用者暱稱 (上限50 字) | String |
| email | True | email | String |
| password | True | 密碼 | String |
| checkPassword | True | 需與登入密碼相同 | String |

---

### Response Header

```
content-type: application/json
```
